# Inception Attack

## How to Run

### Prerequisites
- Python 3.8 or higher
- Required libraries: `easyjailbreak`, `torch`, `transformers`, `pandas`

### Installation
Install required packages:
```bash
pip install easyjailbreak torch transformers pandas
```

### Running the Attack

**Basic Command:**
```bash
python Inception.py <csv_file_path>
```

**Example:**
```bash
python Inception.py /path/to/your/prompts.csv
```

### Input Format
- CSV file with a column containing prompts
- Supported column names: `Original_prompt`, `prompt`, `query`, `goal`, `harmful_prompt`, `text`
- If none of these exist, the first column will be used

### Output
- Generates a CSV file with two columns:
  - `original_prompt`: The input prompt
  - `mutated_prompt`: The mutated prompt generated by the attack
- Output file location: `<input_file_name>_inception_results.csv` in the Sample folder

### Example Usage
```bash
python Inception.py /Users/abhishek/Desktop/ML_Prompt_mutation/ML_Custom_Attacks/Sample/Original_prompts.csv
```

This will create `/Users/abhishek/Desktop/ML_Prompt_mutation/ML_Custom_Attacks/Sample/Original_prompts_inception_results.csv`
